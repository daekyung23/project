자율 주행 자동차


  2. 자율 주행 단계

자율 주행 단계 설명
0단계 - 비자동화 운전자가 차량의 운전 및 속도 제어를 모두 담당하는 단계
자율주행 기능 미포함
1단계 - 운전자 보조 특정 주행 모드에서 차량 조향 또는 감속, 가속 중 하나를 수행하는 단계
시스템이 차량의 속도, 제동을 제어하고 일정 속도를 유지하는 기능 포함
2단계 - 부분 자동화 운전자의 개입 없이 시스템이 자동차의 속도와 방향을 동시에 제어하는 단계
시스템이 스스로 방향을 바꾸거나 간격 유지를 위해 속도제어를 하는 기능 포함
운전자의 전방 주시 필수
3단계 - 조건부 자동화 자율 주행 중 돌발 상황 발생 시에만 운전자가 개입을 해야 하는 단계
시스템이 스스로 추월하거나 장애물을 감지하고 피할 수 있는 기능 포함
시스템이 사고나 교통 혼잡을 감지해 피할 수 있는 기능 포함
4단계 - 고도 자동화 자율 주행이 모든 상황을 대응하는 단계
시스템이 운행 구간 전체를 감시 할 수 있는 기능 포함
운전자는 출발 전 목적지와 이동 경로만 입력
5단계 - 완전 자동화 운전자가 필요 없는 단계
탑승자가 목적지를 말하면 시스템이 판단해서 움직이는 기능 포함
자동차의 기능을 조작할 필요가 없음
