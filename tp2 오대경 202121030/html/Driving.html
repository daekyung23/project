Machine Learning & Deep Learning

Machine Learning

 패턴 인식 및 학습에 사용되는 몇 가지 베이지안 기법에 주로 적용되는 기법
 데이터를 학습하고 최적화하며, 이를 기반으로 예측하고 최종적으로 데이터를 간결한 설명으로 분류할 수 있는 알고리즘의 형태
 명시적 프로그래밍으로 접근하기 어렵거나, 실용성이 없는 경우 주로 사용됨

Deep Learning

 딥러닝은 음성 인식, 이미지 식별 또는 예측 등 사람의 작업을 대신 수행하도록 컴퓨터를 학습시키는 일종의 머신 러닝
 데이터에 대한 기본 파라미터를 설정하고 컴퓨터가 여러 처리 계층을 이용해 패턴을 인식함으로써 스스로 학습하도록 훈련시키는 기술
